Copy Button Demo

This demonstrates the copy button functionality that has been added to code blocks in the WordPress AI Assistant chatbot.

PHP Code Block Example

PHP 
function create_custom_post_type() {
    register_post_type('book',
        array(
            'labels' => array(
                'name' => 'Books',
                'singular_name' => 'Book'
            ),
            'public' => true,
            'has_archive' => true,
            'supports' => array('title', 'editor', 'thumbnail')
        )
    );
}
add_action('init', 'create_custom_post_type');

JavaScript 
function handleFormSubmit(event) {
    event.preventDefault();
    
    const formData = new FormData(event.target);
    const data = Object.fromEntries(formData);
    
    // Send data to server
    fetch('/api/submit', {
        method: 'POST',
        headers: {
            'Content-Type': 'application/json',
        },
        body: JSON.stringify(data)
    })
    .then(response => response.json())
    .then(result => {
        console.log('Success:', result);
    })
    .catch(error => {
        console.error('Error:', error);
    });
}

How It Works in the Chatbot

When the AI responds with code blocks (like the examples above), each code block will automatically:

  1. Display a header showing the programming language
  2. Show a copy button in the top-right corner
  3. Copy the entire code content when clicked
  4. Provide visual feedback that the copy was successful

Note: Inline code (like wp_insert_post()) will not show copy buttons - only multi-line code blocks.
